# Multiple House Teleports

A World of Warcraft addon for saving and teleporting to multiple housing plot locations in Midnight.

## Features

- Save multiple housing plot locations
- One-click macro creation for each location
- Home teleport macro for your own house
- Displays neighborhood type (Public, Guild, Charter) and name
- Clean options panel integrated with WoW Settings

## Installation

1. Download and extract to `World of Warcraft/_retail_/Interface/AddOns/`
2. Restart WoW or `/reload`

## Usage

1. Visit a housing plot
2. Open options with `/mht` or through Settings
3. Click **Add Current Location**
4. Click **Create Macro** and drag it to your action bar

## Commands

| Command | Description |
|---------|-------------|
| `/mht` | Show commands |
| `/mht add [name]` | Add current location |
| `/mht list` | List saved locations |
| `/mht delete <#>` | Delete a location |
| `/mht options` | Open options panel |
